# Weather Telegram Bot — прогноз погоды от Яндекс.Погоды

Телеграм-бот, который показывает текущую погоду, прогноз на сегодня и на три дня, а также присылает ежедневную рассылку в указанное время. Использует API Яндекс.Погоды.

## Возможности

- 🏙 Погода сейчас (`/now`)
- 📅 Прогноз на сегодня (`/today`)
- 📆 Прогноз на 3 дня (`/days3`)
- 🔔 Подписка на утреннюю рассылку (`/subscribe`, `/unsubscribe`)
- 📝 Обработка текстовых запросов: «сейчас», «сегодня», «3 дня»
- ⏰ Рассылка прогноза всем подписчикам каждый день (по умолчанию 8:00 МСК)

## Требования

- Python 3.10+
- Docker (опционально)
- Токен Telegram-бота (получить у [@BotFather](https://t.me/BotFather))
- API-ключ Яндекс.Погоды (получить в [Яндекс.Облаке](https://cloud.yandex.ru/) или на [developer.tech.yandex.ru](https://developer.tech.yandex.ru/))

## Установка и настройка

### 1. Получение токена Telegram-бота

1. Напишите [@BotFather](https://t.me/BotFather) команду `/newbot`
2. Придумайте имя и username бота
3. Скопируйте полученный токен (например, `7123456789:AAH...`)

### 2. Получение API-ключа Яндекс.Погоды

1. Зарегистрируйтесь в [Яндекс.Облаке](https://cloud.yandex.ru/)
2. Создайте сервисный аккаунт и получите API-ключ для доступа к [Yandex Weather API](https://yandex.ru/dev/weather/)
3. Скопируйте ключ (формат UUID)

### 3. Настройка переменных окружения

Скопируйте файл `.env.example` в `.env` и заполните его:

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```ini
TELEGRAM_BOT_TOKEN=ваш_токен
YANDEX_WEATHER_API_KEY=ваш_ключ
CITY_LAT=45.04      # широта города (Краснодар по умолчанию)
CITY_LON=38.98      # долгота города
CITY_NAME=Краснодар
MORNING_HOUR=8      # час рассылки (МСК)
MORNING_MINUTE=0
# MORNING_CHAT_ID=   # необязательно, можно указать конкретный chat_id
```

### Запуск бота

#### Локально (без Docker)

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите бота:

```bash
python bot.py
```

#### С использованием Docker

Соберите образ и запустите контейнер:

```bash
docker-compose up -d
```

Бот будет автоматически перезапускаться при падении. Файл `subscribers.json` будет сохраняться на хосте, так что данные подписчиков не пропадут после перезапуска.

## Команды бота

| Команда        | Описание                           |
| -------------- | ---------------------------------- |
| `/start`       | Приветствие и справка              |
| `/now`         | Погода прямо сейчас                |
| `/today`       | Прогноз на сегодня (по частям дня) |
| `/days3`       | Прогноз на три дня                 |
| `/subscribe`   | Подписаться на утреннюю рассылку   |
| `/unsubscribe` | Отписаться от рассылки             |

Также можно писать текстовые сообщения:

- «сейчас» / «now» / «погода» → `/now`
- «сегодня» / «today» / «прогноз» → `/today`
- «3 дня» / «три дня» / «3days» → `/days3`
- «подписка» / «подписаться» → `/subscribe`
- «отписаться» → `/unsubscribe`

## Примечания

- Бот использует часовой пояс **Москва (UTC+3)** для планирования рассылки.
- Если указан `MORNING_CHAT_ID`, то этот чат будет получать рассылку даже без подписки (для тестирования).
- Файл `subscribers.json` автоматически создаётся при первой подписке. При использовании Docker он монтируется как volume, чтобы данные сохранялись между перезапусками.
- Для изменения города достаточно изменить `CITY_LAT`, `CITY_LON`, `CITY_NAME` в `.env` и перезапустить бота.
